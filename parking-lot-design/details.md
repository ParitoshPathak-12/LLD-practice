# Problem Statement
Design a parking lot having n floors and m spots per floor. It should support
parking vehicle, unparking vehicle, view available spots, view occupied
spots, search a vehicle, etc.

## Supported Commands:
1. park <vehicle_number> <vehicle_type>
2. unpark <vehicle_number>
3. show_available_spots
4. search <vehicle_number>
5. exit
6. help
